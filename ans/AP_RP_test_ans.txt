#### 등차수열 문제 풀이

1. 첫항 3, 공차 5인 등차수열의 첫 5개 항
   a₁ = 3
   a₂ = 8
   a₃ = 13
   a₄ = 18
   a₅ = 23
   답: 3, 8, 13, 18, 23

2. 4, 9, 14, 19, ...의 공차
   연속된 두 항의 차: 9-4 = 5, 14-9 = 5
   답: 5

3. a₃ = 11, a₇ = 23일 때 공차
   23 - 11 = 4d (a₇ - a₃ = 4d)
   12 = 4d
   답: d = 3

4. a₁ = 5, d = 3일 때 a₁₅
   aₙ = a₁ + (n-1)d
   a₁₅ = 5 + (14)3 = 5 + 42
   답: 47

5. 3, 8, 13, 18, ...의 일반항
   a₁ = 3, d = 5
   aₙ = a₁ + (n-1)d = 3 + (n-1)5 = 5n - 2
   답: aₙ = 5n - 2

6. a₄ = 13, a₈ = 29일 때 일반항
   29 - 13 = 4d
   16 = 4d
   d = 4
   a₄ = a₁ + 3d = 13
   a₁ = 13 - 12 = 1
   aₙ = 1 + (n-1)4 = 4n - 3
   답: aₙ = 4n - 3

7. 2, 7, 12, 17, ...의 20번째 항
   a₁ = 2, d = 5
   a₂₀ = 2 + (19)5 = 2 + 95
   답: 97

8. a₁ = 4, d = 6일 때 40이 몇 번째 항
   40 = 4 + (n-1)6
   36 = 6(n-1)
   n = 7
   답: 7

9. a₁ = 3, d = 4일 때 51이 몇 번째 항
   51 = 3 + (n-1)4
   48 = 4(n-1)
   n = 13
   답: 13

10. 2, 6, 10, 14, ...에서 100보다 큰 첫 항
    aₙ = 2 + (n-1)4
    100 < 2 + (n-1)4
    98 < 4(n-1)
    24.5 < n-1
    n > 25.5
    답: 26번째 항

11. a₅ = 17, a₉ = 33일 때 첫째항
    33 - 17 = 4d
    16 = 4d
    d = 4
    17 = a₁ + 4d
    17 = a₁ + 16
    a₁ = 1
    답: 1

12. -2, 3, 8, 13, ...에서 73인 항의 번호
    d = 5
    73 = -2 + (n-1)5
    75 = 5(n-1)
    n = 16
    답: 16

13. a₁ = 5, d = 4인 수열의 첫 10항의 합
    S₁₀ = 10(2×5 + (10-1)×4)/2
    = 10(10 + 36)/2
    = 10×23
    답: 230

14. a₆ = 22, a₁₀ = 38일 때 a₁₅
    38 - 22 = 4d
    16 = 4d
    d = 4
    22 = a₁ + 5d
    22 = a₁ + 20
    a₁ = 2
    a₁₅ = 2 + 14×4 = 58
    답: 58

15. 1, 7, 13, 19, ...에서 연속된 세 항의 합이 63
    d = 6
    aₖ + a(k+1) + a(k+2) = 63
    (1 + (k-1)6) + (1 + k×6) + (1 + (k+1)6) = 63
    3 + (3k+0)6 = 63
    3 + 18k = 63
    18k = 60
    k = 10/3
    k는 자연수여야 하므로 이 조건을 만족하는 세 항은 19, 25, 31
    답: 19, 25, 31

16. a₁ = 3, d = 2일 때 50을 넘는 첫 항의 번호
    50 < 3 + (n-1)2
    47 < 2(n-1)
    23.5 < n-1
    n > 24.5
    답: 25

17. a₂ = 7, a₈ = 25일 때 a₁₂
    25 - 7 = 6d
    18 = 6d
    d = 3
    7 = a₁ + d
    a₁ = 4
    a₁₂ = 4 + 11×3 = 37
    답: 37

18. a₁ = 1, d = 3일 때 첫 n항의 합이 209
    209 = n(2 + (n-1)3)/2
    418 = n(2 + 3n - 3)
    418 = n(3n - 1)
    3n² - n - 418 = 0
    답: n = 12

19. 4, 11, 18, 25, ...에서 두 자리 수의 개수
    10 ≤ 4 + (n-1)7 ≤ 99
    6 ≤ 7(n-1) ≤ 95
    0.857.. ≤ n-1 ≤ 13.571..
    n은 2부터 14까지
    답: 13개

20. a₅ = 14, a₁₀ = 29일 때
    29 - 14 = 5d
    15 = 5d
    d = 3
    14 = a₁ + 4×3
    14 = a₁ + 12
    a₁ = 2
    답: d = 3, a₁ = 2

#### 등비수열 문제 풀이

1. a₁ = 2, r = 3인 등비수열의 첫 5개 항
   2, 6, 18, 54, 162
   답: 2, 6, 18, 54, 162

2. 3, 9, 27, 81, ...의 공비
   9/3 = 3, 27/9 = 3
   답: 3

3. a₂ = 6, a₅ = 162일 때 공비
   162 = 6r³
   27 = r³
   답: r = 3

4. a₁ = 3, r = 2일 때 일반항
   aₙ = 3×2^(n-1)
   답: aₙ = 3×2^(n-1)

5. a₁ = 2, a₄ = 54일 때 일반항
   54 = 2r³
   27 = r³
   r = 3
   aₙ = 2×3^(n-1)
   답: aₙ = 2×3^(n-1)

6. 2, 6, 18, 54, ...의 7번째 항
   a₁ = 2, r = 3
   a₇ = 2×3⁶ = 1458
   답: 1458

7. a₁ = 1, r = 3일 때 729는 몇 번째 항
   729 = 1×3^(n-1)
   729 = 3^(n-1)
   n = 7
   답: 7

8. a₁ = 4, r = 2일 때 256은 몇 번째 항
   256 = 4×2^(n-1)
   64 = 2^(n-1)
   n = 7
   답: 7

9. a₁ = 2, r = 5일 때 1000을 넘는 첫 항의 번호
   1000 < 2×5^(n-1)
   500 < 5^(n-1)
   log₅500 < n-1
   n > 4.86
   답: 5

10. 3, 6, 12, 24, ...의 첫 5항의 합
    Sₙ = a₁(r^n-1)/(r-1)
    = 3(2⁵-1)/(2-1)
    = 3×31
    답: 93

11. a₃ = 18, a₆ = 486일 때 공비
    486 = 18r³
    27 = r³
    답: r = 3

12. a₁ = 5, r = 2인 등비수열의 첫 6항의 합
    S₆ = 5(2⁶-1)/(2-1)
    = 5×63
    답: 315

13. a₁ = 3, a₃ = 48일 때 a₅
    48 = 3r²
    r = 4
    a₅ = 3×4⁴ = 768
    답: 768

14. 1, 4, 16, 64, ...에서 연속된 세 항의 곱이 1024
    r = 4
    aₖ×a(k+1)×a(k+2) = 1024
    1×4^(k-1)×4^k×4^(k+1) = 1024
    4^(3k) = 1024
    3k = 5
    k는 자연수여야 함
    답: 4, 16, 64

15. a₁ = 2, r = 3일 때 1000을 넘는 첫 항의 번호
    1000 < 2×3^(n-1)
    500 < 3^(n-1)
    log₃500 < n-1
    n > 6.26
    답: 7

16. a₄ = 16, a₇ = 128일 때 첫째항
    128 = 16r³
    r = 2
    16 = a₁×2³
    a₁ = 2
    답: 2

17. a₁ = 1, r = 2인 등비수열의 첫 8항의 합
    S₈ = (2⁸-1)/(2-1)
    = 255
    답: 255

18. a₂ = 12, a₅ = 324일 때 a₇
    324 = 12r³
    27 = r³
    r = 3
    12 = a₁r
    a₁ = 4
    a₇ = 4×3⁶ = 2916
    답: 2916

19. 2, 4, 8, 16, ...에서 세 자리 이하 자연수의 개수
    2^(n-1) ≤ 999
    n-1 ≤ log₂999
    n ≤ 10.96
    답: 10개
